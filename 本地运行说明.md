### 方式 2: 用 Python 启动本地服务器

```bash
# 打开终端/命令行
cd /home/admin/.openclaw/workspace/projects/pixel-rpg
python3 -m http.server 8081

# 然后在浏览器访问
http://localhost:8081
```

1. 用 VS Code 打开 `pixel-rpg` 文件夹

3. 右键 `index.html` → "Open with Live Server"
