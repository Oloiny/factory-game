# 🏢 大厂风云 - 职场抓马生存战

> 情绪（Sanity）与进度（Progress）的走钢丝游戏

**版本**: MVP 0.1  
**创建时间**: 2026-03-11  
**技术栈**: Phaser 3.60 + HTML5

---

## 🎮 游戏简介

你是一家大厂的项目经理，需要在**30 天内**完成一个项目。

但最核心的资源不是金币，而是**员工的情绪值（Sanity）**！

- 员工干活会涨**进度**，但会掉**情绪**
- 情绪太低会进入**异常状态**：摸鱼、暴躁、跑路
- 你的博弈：**压榨 vs 安抚**

## 🎯 核心玩法

### 资源系统

| 资源 | 说明 | 恢复方式 |
|------|------|---------|
| **📊 项目进度** | 0-1000，达到 1000 获胜 | 员工工作 |
| **😰 员工情绪** | 0-100，太低会异常 | 休息、团建 |
| **😴 员工疲劳** | 0-100，太高效率低 | 休息、休假 |
| **💰 公司预算** | 用于技能 | 不会自动恢复 |
| **⚡ 管理点数** | 用于技能 | 每天 +10 |

### 员工状态

| 情绪值 | 状态 | 效果 |
|--------|------|------|
| >30 | 😊 正常 | 正常工作 |
| 15-30 | 🐟 摸鱼 | 停止产出进度 |
| 5-15 | 😠 暴躁 | AOE 传染负能量 |
| <5 | 🚪 跑路 | 3 天后离职，带走进度 |

## 🃏 技能卡牌

### 🫓 画大饼（⚡10 MP）
选定一个员工，说一段话。
- **成功**（基于忠诚度）：鸡血 3 天，产出×1.5
- **失败**：忠诚度 -20，还会传八卦

### 🍕 团建聚餐（💰50 预算）
全员聚餐，清空负面情绪。
- 情绪 +30，疲劳 -50
- 有概率消除敌对关系
- **消耗 1 天时间**

### 👔 空降高管（💰200 预算）
招募卷王高管。
- 进度产出×3
- 但周围员工获得"不公平感"Debuff

### 🏖️ 强制休假（💰20 预算）
强制指定员工休假。
- 情绪 +20，疲劳 -30
- **消耗 1 天时间**

### 📢 紧急会议（⚡30 MP）
召集所有人开会。
- 每人进度 +10
- 但情绪 -15

## 📅 游戏流程

1. **游戏开始** - 4 名初始员工（程序×2, PM, 设计）
2. **每分钟 tick** - 员工自动工作，情绪变化
3. **每天检查** - 随机触发危机事件
4. **玩家干预** - 使用技能卡牌管理
5. **胜负判定**：
   - ✅ **胜利**：进度达到 1000
   - ❌ **失败**：死线（30 天）到了但进度<80%
   - ❌ **失败**：所有人都离职

## 🎲 随机事件

### 情绪崩溃
> 核心程序员在工位上突然崩溃大哭

**选项**:
- 报警开除 → 他走了，进度 -10%
- 知心开导 → 消耗 20MP，情绪 +20
- 给一天假 → 消耗¥20，情绪 +15

### 竞品挖角
> 竞品公司正在高薪挖角你的主策划

**选项**:
- 画更大的饼 → 消耗 30MP，60% 概率留住
- 加薪挽留 → 消耗¥100，必留住
- 让他走 → 他离职了，进度 -20%

### 会议室吵架
> 两个员工在会议室吵起来了

**选项**:
- 让他们继续吵 → 两人情绪 -10，关系恶化
- 介入调解 → 消耗 15MP，暂时平息
- 请奶茶 → 消耗¥40，全员情绪 +10

### 裁员谣言
> 吸烟区传出公司要裁员的谣言

**选项**:
- 不回应 → 全员情绪 -15，忠诚 -10
- 发全员邮件 → 消耗 20MP，稳定军心
- 开大会安抚 → 消耗 30MP+¥50，成功稳定
